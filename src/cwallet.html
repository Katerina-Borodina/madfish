Cryptocurrency Wallet

Cryptocurrency wallet, supporting several major currencies

The challenge

A cryptocurrency startup required a wallet with the most user-friendly interface, transparency and support for the main cryptocurrencies, ERC20-tocurrencies, as well as the cryptocurrency of the startup itself.

The solution

A cross-platform solution was developed using Electron.js technology, which allows writing cross-platform desktop applications for MacOs and Windows operating systems. Also, the functionality of atomic swap (link) was implemented, allowing the exchange of ERC20 tokens for the main cryptocurrency of the company.
