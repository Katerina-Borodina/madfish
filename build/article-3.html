How to Create UI Animations That Meet Users’ Needs

Vita Sporevaya

UX/UI Designer

What’s the ColorMatchTab animation about?

There are multiple ways to organize navigation in apps: tab bars, side menus, Tinder-like swipes between screens. An issue with most existing solutions, however, is that they are inconvenient for phones with large screens. Users have to constantly stretch their thumbs to the icon to go to the next screen.

We decided to share the user interface animation concept we created that solves this problem of in-app navigation on large screens. This animation was developed to illustrate a concept we developed for a review app. The review app would show users interesting places around them and let them.
